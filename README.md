# 🚀 Resume Builder AI

> **Build ATS-friendly, professional resumes effortlessly with the power of Generative AI.**

A modern **MERN Stack** application that helps users create, customize, preview, and download high-quality resumes in minutes. The application leverages **Generative AI** to generate professional summaries, improve resume content, and streamline the resume creation process.

---

## ✨ Features

* 🤖 AI-powered resume generation
* 📄 ATS-friendly resume templates
* 👤 Secure JWT authentication
* ✍️ Live resume editing & preview
* 📥 One-click PDF download
* 📱 Fully responsive design
* ⚡ Fast and intuitive user experience

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Context API
* Axios

**Backend**

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

**AI**

* Google Gemini API

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Himanshu-000/resume-builder-ai.git

# Install dependencies
npm install

# Start the application
npm run dev
```
