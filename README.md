# 🚀 Resume Builder AI

> **An AI-powered Resume Builder that enables users to create ATS-friendly, professional resumes with the power of Generative AI.**

Resume Builder AI is a full-stack **MERN** application designed to simplify the resume creation process. It allows users to build, customize, preview, and download professional resumes while leveraging **Google Gemini AI** to generate impactful content, optimize resume sections, and improve overall quality.

The application focuses on delivering a seamless user experience with secure authentication, responsive design, and downloadable ATS-compatible resumes.

---

## ✨ Key Features

- 🤖 AI-powered resume content generation using Google Gemini API
- 📄 ATS-friendly and professionally designed resume templates
- 👤 Secure user authentication and authorization with JWT
- ✍️ Real-time resume editing with instant preview
- 📥 Export resumes as high-quality PDF documents
- 📱 Fully responsive interface for desktop, tablet, and mobile devices
- ⚡ Fast, intuitive, and user-friendly experience

## 📂 Project Setup

### Clone the Repository

```bash
git clone https://github.com/Himanshu-000/resume-builder-ai.git
cd resume-builder-ai
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

The application will start on your local development server.
